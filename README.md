# Mezion - Location-Based Lost & Found Platform 🗺️🔍

Mezion is a comprehensive Android application designed to help users track, report, and find lost items using interactive maps and real-time communication. 

## 📖 Project Documentation
**[Click here to view the full Project Book (PDF)](./Mezion_Project_Book.pdf)**
*I highly recommend checking out the project book for an in-depth look at the system architecture, design decisions, and UI/UX flows.*

## ✨ Key Features
* **Interactive Map Integration:** Uses Google Maps API for geospatial queries and radius-based filtering to locate items near the user.
* **Real-Time Chat:** An instant messaging system with media support, allowing finders and owners to communicate seamlessly.
* **Secure Authentication:** User management and secure login powered by Firebase Authentication.
* **Live Database:** Real-time data synchronization and persistent storage utilizing Firebase Realtime Database.

## 🛠️ Tech Stack
* **Frontend:** Java, XML (Android Studio)
* **Backend & Database:** Firebase (Authentication, Realtime Database, Storage)
* **APIs:** Google Maps API
* **Architecture:** Full-stack MVC oriented structure

## 🚀 Getting Started
To view or run the project locally:
1. Clone the repository: `git clone https://github.com/YourUsername/Mezion.git`
2. Open the project in Android Studio.
3. Sync the Gradle files and ensure you have a valid `google-services.json` file configured with Firebase.
4. Build and run the app on an emulator or physical device.
